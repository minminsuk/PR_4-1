## 📌 제목 (Headings)

```markdown

## H2 제목
### H3 제목
#### H4 제목
##### H5 제목
###### H6 제목
```

## H2 제목
### H3 제목
#### H4 제목
##### H5 제목
###### H6 제목

## ✍️ 텍스트 서식 (Text Formatting)

```markdown
**굵게 (Bold)**
*기울임 (Italic)*
~~취소선 (Strikethrough)~~
**_굵고 기울임_**
`인라인 코드`
<u>밑줄 (Underline)</u>
```

| 문법 | 결과 |
|------|------|
| `**굵게**` | **굵게** |
| `*기울임*` | *기울임* |
| `~~취소선~~` | ~~취소선~~ |
| `**_굵고 기울임_**` | **_굵고 기울임_** |
| `` `인라인 코드` `` | `인라인 코드` |
| `<u>밑줄</u>` | <u>밑줄</u> |

## 📋 목록 (Lists)

### 순서 없는 목록 (Unordered List)

```markdown
- 항목 1
- 항목 2
  - 중첩 항목 2-1
  - 중첩 항목 2-2
- 항목 3
```

- 항목 1
- 항목 2
  - 중첩 항목 2-1
  - 중첩 항목 2-2
- 항목 3

### 순서 있는 목록 (Ordered List)

```markdown
1. 첫 번째
2. 두 번째
   1. 중첩 2-1
   2. 중첩 2-2
3. 세 번째
```

1. 첫 번째
2. 두 번째
   1. 중첩 2-1
   2. 중첩 2-2
3. 세 번째

## 🔗 링크 & 이미지 (Links & Images)

```markdown
<!-- 링크 -->
[링크 텍스트](https://example.com)
[링크 텍스트](https://example.com "툴팁 텍스트")

<!-- 이미지 -->
![대체 텍스트](이미지URL)
![대체 텍스트](이미지URL "툴팁 텍스트")

<!-- 이미지에 링크 걸기 -->
[![대체 텍스트](이미지URL)](링크URL)
```

- 링크 예시: [GitHub](https://github.com)
- 이미지 예시: `![로고](https://example.com/logo.png)`

## 💬 인용문 (Blockquote)

```markdown
> 1단계 인용문
>> 2단계 인용문
>>> 3단계 인용문
```

> 1단계 인용문
>> 2단계 인용문
>>> 3단계 인용문

> 💡 **Tip:** 인용문 안에도 다른 마크다운 문법을 사용할 수 있습니다.

## 📊 표 (Table)

```markdown
| 헤더1 | 헤더2 | 헤더3 |
|-------|:-----:|------:|
| 좌측 정렬 | 가운데 정렬 | 우측 정렬 |
| 내용 | 내용 | 내용 |
```

| 헤더1 | 헤더2 | 헤더3 |
|-------|:-----:|------:|
| 좌측 정렬 | 가운데 정렬 | 우측 정렬 |
| 내용 A | 내용 B | 내용 C |
| 내용 D | 내용 E | 내용 F |

> `:---` = 좌측 정렬 / `:---:` = 가운데 정렬 / `---:` = 우측 정렬

## 😄 이모지 (Emoji)

| **상태** | ⚠️ | 주의 / 경고 |
| **상태** | 🔄 | 진행 중 / 업데이트 |
| **상태** | 🚧 | 작업 중 (WIP) |
| **정보** | 💡 | 팁 / 아이디어 |
| **정보** | 📌 | 중요 포인트 |
| **정보** | 📝 | 노트 / 메모 |
| **정보** | 📖 | 문서 / 가이드 |
| **정보** | 🔍 | 검색 / 조사 |
| **개발** | 💻 | 코드 / 개발 |
| **개발** | 🐛 | 버그 |
| **개발** | 🔧 | 수정 / 설정 |
| **개발** | 🚀 | 배포 / 출시 |
| **개발** | 📦 | 패키지 / 라이브러리 |
| **개발** | 🗂️ | 폴더 / 파일 |

## 🧩 기타 팁

- 줄바꿈은 문장 끝에 **스페이스 2칸** 또는 **빈 줄 하나** 추가
- `<!-- 주석 -->` 으로 마크다운 안에 주석 작성 가능
- HTML 태그도 마크다운 안에서 사용 가능 (예: `<br>`, `<u>`, `<details>`)

### 접기/펼치기 (Details)

```markdown
<details>
<summary>클릭하여 펼치기 👇</summary>

숨겨진 내용이 여기에 들어갑니다.

</details>
```

<details>
<summary>클릭하여 펼치기 👇</summary>

숨겨진 내용이 여기에 들어갑니다. 코드나 긴 설명을 숨길 때 유용해요!

</details>
